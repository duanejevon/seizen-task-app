# 1. FluentDesignTokens

**Goal**: Establish Windows 11/Fluent-inspired design tokens and restyle existing chrome to match — no background image, translucency, or menu changes yet.
**Est.**: 1.5h
**Dependencies**: none

## Steps
- [x] Define new CSS custom properties for a Fluent-style palette (softer accent, neutral surfaces), Fluent-typical corner radii, and softer multi-layer shadows
- [x] Update buttons, inputs, board switcher, column headers, and cards to use the new tokens (no structural changes)
- [x] Verify existing functionality (build, smoke tests) unaffected — purely visual
- [x] Screenshot and informally compare the new look against the MS To Do reference for general direction

## Definition of Done
- [x] New token set in `index.css` fully replaces the old palette where applicable
- [x] All existing UI elements (buttons, inputs, sidebar, columns, cards) are visually consistent with the new tokens
- [x] `npm run build` passes; `smoke:db`/`smoke:crash` unaffected
- [x] Screenshot reviewed informally against the Win11/Fluent direction
