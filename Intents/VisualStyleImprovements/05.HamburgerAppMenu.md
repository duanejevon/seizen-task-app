# 5. HamburgerAppMenu

**Goal**: Hide Electron's native menu bar and add an in-app hamburger button/menu that currently houses the background picker entry point.
**Est.**: 1.5h
**Dependencies**: 04

## Steps
- [ ] Set `Menu.setApplicationMenu(null)` (or equivalent) in `electron/main.js` to remove the native File/Edit/View/Window/Help bar
- [ ] Add a hamburger icon button to the app shell
- [ ] Build a simple dropdown/panel menu triggered by the hamburger button
- [ ] Move the Intent 4 background picker entry point into this menu
- [ ] Structure the menu component so future settings entries can be added without rework

## Definition of Done
- [ ] Native menu bar no longer appears
- [ ] Hamburger button opens an app menu containing the background picker
- [ ] Board sidebar remains always visible (unaffected by the hamburger menu)
