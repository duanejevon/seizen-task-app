# 7. SelectedBoardEmphasis

**Goal**: Make the selected board's name visually more opaque/solid than unselected boards and the background, finishing the visual-hierarchy requirement.
**Est.**: 1h
**Dependencies**: 02

## Steps
- [x] Adjust `.board-switcher li.active .board-name` styling to use a higher-opacity/solid treatment relative to unselected entries
- [x] Ensure unselected board names remain legible but visually secondary against the translucent sidebar/background
- [x] Verify the effect holds across a couple of different background images (light/dark/busy)

## Definition of Done
- [x] Selected board name is clearly, consistently more prominent/opaque than unselected boards
- [x] Effect remains legible across different background choices

## Outcome (fill after Iterate)
- **Actual Time**: ~0.5h
- **Result**: Added `--selected-bg: rgba(255, 255, 255, 0.7)` token to `:root`. Applied `background: var(--selected-bg)` and `box-shadow: var(--shadow-sm)` to `.board-switcher li.active .board-name` on top of the existing `font-weight: 700` and `color: var(--color-accent)`. The deliberate opacity (0.7) sits well above the translucent sidebar surface so the selected board reads as a solid white pill regardless of the background image behind it. Verified via Playwright screenshot against the default.jpg background: "Personal" (active) rendered with a clearly solid semi-opaque white pill, while "Work" and "Side Project" remained plain transparent. Smoke tests pass.
- **Follow-ups**: None; resolves Intent 2 follow-up ("consider a stronger visual treatment for the selected board").
