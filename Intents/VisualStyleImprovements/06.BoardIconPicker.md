# 6. BoardIconPicker

**Goal**: Per-board icon support — emoji picker plus custom image upload, shown next to the board name.
**Est.**: 2h
**Dependencies**: 03

## Steps
- [ ] Add an `icon` column to the boards table (stores either an emoji character or a reference to a copied custom image file)
- [ ] Extend board IPC handlers to support setting/updating a board's icon
- [ ] Build an emoji picker UI accessible from each board's row in the sidebar
- [ ] Add a "custom image" option that reuses the Intent 3 image-copy pattern (copy into `userData/board-icons/`)
- [ ] Render the icon (emoji or image) next to the board name in the sidebar

## Definition of Done
- [ ] User can assign an emoji or a custom image as a board's icon
- [ ] Icon displays next to the board name and persists across restarts
- [ ] Existing boards without an icon set still render correctly (sensible default/fallback)
