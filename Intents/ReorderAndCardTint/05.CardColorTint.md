# 5. CardColorTint

**Goal**: Give each card a subtle background tint matching its assigned color, in addition to the existing left-border accent.
**Est.**: 1h
**Dependencies**: none

## Steps
- [x] In `src/components/CardItem.tsx`, compute a low-opacity background from `card.color` (e.g. convert the hex to an `rgba(...)` with ~12% alpha) and apply it as `backgroundColor` alongside the existing `borderLeftColor: card.color` (CardItem.tsx:31).
- [x] Apply the same tint treatment to the `DragOverlay` card preview in `src/components/KanbanBoard.tsx` (currently only sets `borderLeftColor`) so the dragged card matches.
- [x] Verify text contrast/readability over the tint for all 8 existing swatches in `CardColorPicker.tsx` — adjust the opacity if any combination is too low-contrast against light/dark theme surfaces.
- [x] Confirm the tint respects the existing translucent/glass surface styling from VisualStyleImprovements (don't fight the blur/opacity tokens already in place).

## Definition of Done
- [x] Every card shows a visible background tint matching its color, with the left-border accent still present.
- [x] Text remains readable over the tint for all 8 swatches.
- [x] Dragged card preview (DragOverlay) shows the same tint.

## Outcome (fill after Iterate)
